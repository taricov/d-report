## About:

- Project - D-Report is a tool for building custom reports that works on top of Daftra ERP.
- Autor - [Taric Ov](https://linkedin.com/in/taricov/)


<br />

## Technologies:

- TailwindCSS
- React
- Mantine 
- React Beautiful DND
- Mantine React Tables 
- HTML
- CSS

## How To Use:

```bash
# install
$ yarn

# watch mode development
$ yarn build

# production mode
$ npm run start:prod
```
